Prototypes

Prototypes for requesting and tracking housing repairs online

Request a repair

Versions 1 and 2 of the prototype were developed and user tested in a previous phase of the project.

Version 3

This version of the prototype combines versions 1 and 2.

It includes the start page, eligibility, emergency repair signposting, detailed problem reporting for "Drip or leak" repairs, providing contact details and information about a vulnerable resident, choosing and booking an appointment slot, confirming the repair has been reported and an appointment has been booked and providing a reference.

Version 4

This version of the prototype includes more detailed problem reporting for the following types of repairs: cupboards or worktop, electrical, heating or hot water. It also includes the ability to report communal repairs.

It has not been tested with any users.
