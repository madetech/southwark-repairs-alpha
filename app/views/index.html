Prototypes

Prototypes for requesting and tracking repairs

Request a repair

Version 1

This version of the protoype includes the start now, eligibility, emergency repair signposting, detailed problem reporting and confirmation parts of the Request a repair service pattern.

It was tested with four residents on the 16th of January 2020.

Version 2

This version of the protoype includes the start now, eligibility, problem reporting, choosing availablity, booking and confirmation parts of the Request a repair service pattern.

It was tested with 4 residents on the 11th of Feburary 2020.

Version 3

This version of the prototype combines the above two versions. It includes the start now, eligibility, emergency repair signposting, detailed problem reporting, choosing availability, and booking and confirmation parts of the Request a repair service pattern.
